## 🚀 Getting Started
To run this project locally:  

```bash
# Clone the repo
git clone https://github.com/your-username/flutter_ui_designs.git

# Navigate to folder
cd flutter_ui_designs

# Get dependencies
flutter pub get

# Run the app
flutter run
